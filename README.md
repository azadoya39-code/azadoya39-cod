1. Краткое описание проекта и его цели
Этот проект представляет собой Telegram-бота, который помогает пользователям формировать полезные привычки. Бот позволяет в интерактивном режиме добавлять цели (например, «читать 30 минут в день»), отмечать их выполнение, просматривать статистику с наглядным прогресс-баром и получать мотивационные сообщения.

Тема курсовой работы: «Разработка Telegram-бота для трекинга привычек с поддержкой интерактивного добавления целей, контроля выполнения, сбора статистики и мотивации пользователей.»

3. Запуск проекта
3.1. Локальный запуск (команды)
pip install -r requirements.txt
python main.py
3.2. Запуск с использованием Docker
docker build -t habit-tracker-bot .
docker run --rm habit-tracker-bot
4. Конфигурация (переменные окружения)
Для работы бота нужен токен:
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
5. Примеры использования
Бот понимает команды:

/start — начало работы

/add — добавить привычку

/done — отметить выполнение

/goals — показать цели

/stats — статистика

/motivation — мотивация

/help — справка

/cancel — отмена
